# AX6-Lean
build Redmi AX6 firmware from https://github.com/Cathgao/openwrt (fork from https://github.com/coolsnowwolf/lede)

**默认超频到2.2GHz，若出现不稳定现象请自行到CPU性能优化中调低最大 Turbo Boost CPU 频率**
